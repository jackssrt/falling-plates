# falling-plates

an unfinished rewrite of a roblox game with [flamework](https://flamework.fireboltofdeath.dev/) and [roblox-ts](https://roblox-ts.com/docs/).  
it's a round based game inspired by minecraft spleef. each round players are spawned into the map and fight to be the last one standing.  
uses roact for its ui. should be updated to normal react xd
